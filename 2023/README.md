My solutions for AOC 2022.

## How to run

Dependencies

- rust (nightly)
- cargo

Run a specific day:

```
cargo run -- run 1
```

Run tests:

```
cargo test
```

Generate template and fetch input for a specific day:

```
cargo run -- prepare 1
```
