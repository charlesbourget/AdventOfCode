My solutions for AOC 2023.

## How to run

Dependencies

- rust (nightly)
- cargo

Run a specific day:

```bash
cargo run -- run -d 1
```

Run all days:

```bash
cargo run -- run --all
```

Run tests:

```bash
cargo test
```

Generate template and fetch input for a specific day:

```bash
cargo run -- prepare 1
```
